## Building + Running

- **bun** runtime (â‰¥1.0.0)

### Development

1. **Install dependencies**

   ```sh
   bun install
   ```

2. **Build TypeScript**

   ```sh
   bun run build
   ```

3. **Run in development mode**

   ```sh
   bun run dev
   ```

4. **Run in production mode**

   ```sh
   bun run prod
   ```

### Lint & Format

- **Lint code:**  

  ```sh
  bun run lint
  ```

- **Format code:**  

  ```sh
  bun run format
  ```
